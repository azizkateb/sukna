# سُكنى — Next.js

واجهة عربية RTL لمنصة عقارات وإيجارات قصيرة المدى، مبنية باستخدام Next.js App Router وTypeScript.

## التشغيل

```bash
npm install
npm run dev
```

افتح `http://localhost:3000`.

## التخصيص

- بيانات العقارات والمدن: `app/page.tsx`
- التصميم والاستجابة: `app/globals.css`
- الصور: روابط Pexels مباشرة بدقة 1080p داخل `app/page.tsx` و`app/globals.css`
- استبدل رقم `966500000000` برقم واتساب الخاص بالعميلة.

الـ Hero مصمم بعرض الشاشة كاملاً، بينما المحتوى الداخلي يبقى داخل حاوية منظمة.
شريط التنقل شفاف ومثبت بصرياً فوق صورة الـ Hero.
الصور مستضافة عبر الإنترنت: Hero وبطاقات العقارات بنسبة 1920×1080، وبطاقات الوجهات بعرض 1080px. يتطلب ظهورها اتصالاً بالإنترنت.
